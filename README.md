# Meeting Mind Map Generator (会议纪要思维导图生成器)

这是一个基于 React 和 AI 的智能工具，旨在将繁杂的会议纪要文本（.txt）一键转化为清晰的**思维导图**和**流程图**。它利用大语言模型（LLM）的分析能力，自动提取关键信息、行动项和时间线，帮助你快速回顾和掌握会议重点。

## ✨ 主要功能

*   **📝 文本转可视化**: 上传会议记录文本文件，自动生成结构化的思维导图。
*   **🔄 流程图生成**: 自动识别会议中的流程、步骤或时间线，并生成可视化的流程图。
*   **💎 富文本卡片**: 支持“富卡片”结构，可展示包含多列子项、对比列表等复杂内容的节点。
*   **🎨 个性化定制**:
    *   **图标颜色**: 支持随机或自定义思维导图卡片的图标颜色。
    *   **模型显示**: 在卡片上显示生成该导图所使用的 AI 模型名称。
*   **⚙️ 灵活的 AI 配置**:
    *   自定义 API Base URL 和 API Key (兼容 OpenAI 格式)。
    *   自定义模型名称 (如 `gemini-3-flash-preview`, `gpt-4o` 等)。
    *   **可编辑的系统提示词 (System Prompt)**: 调整 AI 的指令以满足特定需求，并支持一键重置为默认的高级提示词。
*   **🗂️ 本地管理**:
    *   自动保存生成的导图历史到本地浏览器存储 (LocalStorage)。
    *   支持按时间排序、搜索和删除导图。
    *   现代化、响应式的 UI 设计 (Tailwind CSS)。

## 🚀 快速开始

### 1. 安装依赖

确保你已经安装了 Node.js。在项目根目录下运行：

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

启动后，在浏览器中访问控制台显示的地址 (通常是 `http://localhost:5173`)。

### 3. 配置 AI

首次使用前，请点击右上角的 **设置 (Settings)** 图标：

1.  输入你的 **API Base URL** (例如: `https://api.openai.com/v1` 或其他兼容服务商地址)。
2.  输入你的 **API Key**。
3.  (可选) 修改 **Model Name** (默认推荐 `gemini-3-flash-preview`)。
4.  点击 **Save Configuration** 保存。

### 4. 生成导图

1.  回到首页。
2.  点击 **New Mind Map** 卡片。
3.  选择一个 `.txt` 格式的会议纪要文件。
4.  等待 AI 分析生成，完成后将自动跳转到详情页。

## 🛠️ 技术栈

*   **前端框架**: React + Vite
*   **样式库**: Tailwind CSS
*   **图标库**: Lucide React
*   **路由**: React Router DOM
*   **Markdown 渲染**: React Markdown

## 📄 提示词 (Prompt)

项目内置了经过优化的系统提示词，专注于生成：
1.  **按时间顺序**排列的节点。
2.  **详细的内容**而非简短的关键词。
3.  **富文本结构** (Rich Card) 以展示复杂信息。

你可以在设置页面随时调整或重置此提示词。
